Algorithm Explanation:-

The Euclidean algorithm is used to calculate the greatest common divisor (GCD) of two natural numbers a and b.

The algorithm states that for two natural numbers a and b, such that a > b and if :-

r0 = a mod b

r1 = b mod r0

r2 = r0 mod r1

..

..

rk = rk-2 mod rk-1

and rk = 0 than GCD of a and b will be rk-1


Approaches might be used for solving the problem

Approach 1:

Before explaining the approach let's see one case of this problem. Consider a and b be two natural numbers such that a > b and b is a factor of a i.e. b mod a = 0. In this case GCD(a,b) will be equal to b.

Now let's consider the case in which b is not a factor of a i.e. a mod b != 0 in this case let GCD(a,b) be g. So g is a natural number which divides both a and b completely. If we consider this problem backward, so g will exactly divide a mod b, as:

a = x*b + c,

So if g divides b completely then it will divide x*b also and since g also divides a completely we can state that :
